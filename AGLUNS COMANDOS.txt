git init = inicia o git

git status = vê o situacao do repositorio  (verde = esta pronto para ser comitado)

$ git add .= add todos arquivos.

git commit -m "descricao do commit"

git log = log dos commits

git clean -df
git checkout -- .  = volta para o ultimo commit

git reset --soft HEAD~1 = Remover o último commit mantendo as alterações nos arquivos:

git reset --hard HEAD~1 =  Remover o último commit INCLUSIVE as alterações nos arquivos (PERIGO!):
